AI 에이전트는 도구는 잘 쓰지만 길을 잃는다 — AAR 벤치마크

AI 에이전트가 도구를 능숙하게 사용하는 것은 맞지만, 정작 '어디서 정보를 찾아야 하는지'는 잘 모른다. 미네소타대학과 그래머리(Grammarly) 연구진이 공개한 새로운 벤치마크 'The Amazing Agent Race(AAR)'는 기존 평가 체계의 심각한 맹점을 폭로하며, AI 에이전트의 진정한 약점이 '탐색(navigation)'에 있음을 밝혔다.

기존 벤치마크의 맹점: 너무 쉬운 시험

AI 에이전트의 성능을 측정하는 벤치마크는 수십 가지가 있지만, 대부분 심각한 한계를 공유한다. 연구진의 분석에 따르면, 기존 벤치마크의 55%100%가 단 25단계의 단순 직선형(linear chain) 작업으로 구성되어 있다.

이는 마치 '직선 도로에서의 운전 실력'만으로 운전 면허를 발급하는 것과 같다. 실제 업무에서 AI 에이전트는 동시에 여러 정보를 조합하고(fork), 다시 하나로 합치는(merge) 복합적 작업을 수행해야 한다.

AAR의 설계: DAG 구조로 현실을 반영하다

AAR 벤치마크는 '방향성 비순환 그래프(DAG)' 구조의 퍼즐로 구성된다. DAG란 한 방향으로만 흐르되 갈래가 나뉘고 합쳐지는 구조를 말한다. 이를 통해 실제 업무의 복잡성을 더 현실적으로 반영한다.

벤치마크의 구성:

  • 순차형(Sequential): 800개 인스턴스 — 단계별로 이전 결과를 활용하는 연쇄 작업
  • 조합형(Compositional): 600개 DAG 인스턴스 — 분기-합류(fork-merge) 구조로, 여러 하위 작업의 결과를 조합해야 하는 복합 작업

모든 인스턴스는 위키피디아 데이터를 기반으로 자동 생성(procedural generation)되어, 데이터 오염(data contamination) 문제를 방지하고 무한한 확장이 가능하다.

또한 연구진은 실패 원인을 정확히 진단하기 위한 세 가지 진단 지표(diagnostic metrics)를 도입했다:

  1. 탐색 오류율(Navigation Error Rate): 정보를 찾는 과정에서의 실패
  2. 도구 사용 오류율(Tool-Use Error Rate): 도구를 잘못 호출하거나 결과를 잘못 해석하는 실패
  3. 산술 오류율(Arithmetic Error Rate): 수치 계산에서의 실패

실험 결과: 도구 사용보다 탐색이 문제

1,400개 인스턴스에 대한 실험 결과는 놀랍다.

  • 최고 성능 에이전트도 정확도 37.2%에 불과 — 현재 AI 에이전트의 복합 작업 수행 능력이 아직 초기 단계임을 시사한다.
  • 탐색 오류가 전체 실패의 27~52%를 차지 — 반면 도구 사용 오류는 17% 미만이다.
  • 모델 크기보다 아키텍처가 중요 — 더 큰 모델이 반드시 더 나은 성능을 보이지 않는다.

핵심 메시지는 명확하다. AI 에이전트는 '도구를 사용하는 법'은 이미 잘 알고 있지만, '어디서 정보를 찾아야 하는지'를 모른다. 기존 벤치마크가 단순 직선형 작업에 집중했기 때문에 이 근본적 약점이 감춰져 있었다.

에이전트 개발 방향에 던지는 질문

AAR의 발견은 AI 에이전트 개발 방향에 중요한 시사점을 제공한다. 현재 많은 연구가 '더 많은 도구 지원', '더 나은 함수 호출(function calling)' 등 도구 사용 능력 향상에 집중하고 있지만, AAR은 정보 탐색 및 검색 능력이 더 시급한 개선 영역임을 보여준다.

또한 벤치마크 설계에 대한 경종이기도 하다. 단순한 벤치마크에서 높은 점수를 받는다고 해서 실제 업무에서 효과적이라는 보장이 없으며, 평가 체계 자체의 복잡성을 높여야 진정한 성능 격차를 발견할 수 있다.

한계점 및 향후 전망

AAR의 모든 인스턴스가 위키피디아 기반이라는 점에서, 특정 도메인(코딩, 과학 연구 등)에서의 탐색 능력을 직접 측정하기는 어렵다. 또한 '탐색'과 '계획'의 경계가 모호할 수 있어, 탐색 오류로 분류된 실패 중 일부는 계획 수립 능력의 부족으로도 해석될 수 있다.

향후 연구로는 에이전트의 정보 탐색 전략을 강화하기 위한 검색 증강 생성(RAG)과의 통합, 그리고 탐색 경로를 학습하는 강화학습 기반 접근이 유망하다.
