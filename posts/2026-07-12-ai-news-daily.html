이번 호는 지난 주말 사이클에서 새로 확정된 프런티어 발표가 없었던 만큼, 아직 데일리에 정리하지 않은 최근 72시간 이내의 주요 모델 출시 두 건을 캐치업으로 다루는 라이트 에디션이다. 두 소식 모두 7월 9일 발표된 항목으로, 발생일을 그대로 표기해 시의성 기준(발생 72시간 이내)에 맞춰 편입했다. 하나는 메타(Meta)의 첫 유료 API 모델, 다른 하나는 바이트댄스(ByteDance)의 전문가용 이미지 생성 모델이다.

편집 노트 — 오늘 다루는 두 항목은 1차 출처(각 사 공식 발표)로 발생일(7월 9일)이 확인된 소식이다. 다만 두 가지는 본문에서 조심스럽게 읽어 주시길 권한다. ① 메타 Muse Spark 1.1의 가격 수치는 메타 공식 페이지에 요율이 명시되지 않아 복수 매체 보도를 근거로 한 것으로, 본문에서 "약", "보도에 따르면" 등으로 표기했다. ② 바이트댄스 Seedream 5.0 Pro의 성능·품질 서술은 아직 독립 검증 전이라 벤더(제조사) 자체 발표에 근거한다. 두 항목 모두 가격·성능 우위를 단정하기보다 기능 사실과 시장 맥락 위주로 정리했다.

여기서 API란 다른 소프트웨어가 해당 AI 모델을 호출해 쓸 수 있게 열어 둔 연결 창구를, 토큰(token)은 AI가 글을 처리하는 최소 단위(대략 단어 조각)를 가리킨다. 가격은 보통 '입력·출력 100만 토큰당 얼마'로 매겨진다.

메타, 첫 유료 모델 'Muse Spark 1.1'과 Meta Model API 공개

카테고리: 제품 출시 · 파운데이션 모델

메타의 초지능 연구 조직 Meta Superintelligence Labs가 자사 첫 유료 API 모델 'Muse Spark 1.1'을 공개하고, 이를 제공하는 Meta Model API를 퍼블릭 프리뷰(공개 시험판)로 열었다. 우선 미국 개발자를 대상으로 하며, 에이전트·코딩 과제에 초점을 둔 모델이다. 여기서 에이전트(agent)란 사람이 매 단계 지시하지 않아도 목표를 향해 스스로 도구를 쓰고 판단을 이어 가는 AI를 말한다. 일반 사용자는 Meta AI 앱과 meta.ai 웹에서 'Thinking(사고)' 모드로도 이 모델을 써 볼 수 있다. 마크 저커버그가 약 3년 만에 X(옛 트위터)에 복귀해 직접 출시를 알린 점도 함께 화제가 됐다.

가격은 눈에 띄는 대목이다. 복수 매체 보도에 따르면 입력은 100만 토큰당 약 1.25달러, 출력은 약 4.25달러 수준이며, 신규 계정에는 20달러 상당의 크레딧이 제공되는 것으로 전해진다. 반복 입력을 저장해 재사용하는 '캐시 입력' 요율은 약 0.15달러로 보도됐다. 매체들은 이를 Claude Opus 4.8(입력 5달러·출력 25달러)이나 GPT-5.5(약 5달러·30달러)의 약 4분의 1 수준으로 소개했다. 다만 정확한 요율은 메타 공식 페이지에 명시되지 않았으므로, 위 수치는 보도를 근거로 한 것임을 감안해 읽을 필요가 있다.

이번 출시의 의미는 메타의 시장 전략 전환에 있다. 그동안 메타는 모델 가중치를 공개하는 '오픈웨이트' 노선에 무게를 실어 왔는데, 이번에는 요금을 받는 프런티어 경쟁 API 시장에 직접 발을 들였다. 보도된 대로 가격이 공격적이라면 에이전트·코딩 워크로드를 둘러싼 단가 경쟁을 자극할 소지가 있다. 실제 성능과 가격의 경쟁력은 개발자들의 실사용 데이터가 쌓이면서 확인될 대목이다.

발생일: 2026년 7월 9일(메타 공식 블로그 기준). 지난 주말(7/11) 사이클에는 확정 발표가 없어, 72시간 이내 미보도 항목으로 이번 호에 편입했다.

바이트댄스, 전문가용 이미지 모델 'Seedream 5.0 Pro' 공개

카테고리: 제품 출시 · 멀티모달 AI

바이트댄스가 전문가용 멀티모달 이미지 생성·편집 모델 Seedream 5.0 Pro를 공개했다. 멀티모달(multimodal)이란 글·이미지 등 여러 형태의 데이터를 함께 다루는 방식을 뜻한다. 바이트댄스가 이 모델에서 내세우는 기능은 크게 네 가지다.

  • 영역 정밀 편집 — 이미지의 특정 지점을 찍거나, 스케치·주석을 더해 원하는 부분만 골라 수정한다.
  • 다국어 텍스트 렌더링 — 10개 이상의 언어로 이미지 안에 글자를 그려 넣는다.
  • 고밀도 인포그래픽 — 정보량이 많은 도표형 이미지를 생성한다.
  • 시네마틱 화질 — 영상 제작 파이프라인에 쓸 만한 영화 같은 화질을 목표로 한다.

모델은 바이트댄스의 크리에이티브 도구 Dreamina(dreamina.capcut.com)와, Volcano Engine·Byteplus ModelArk·fal.ai 같은 API 파트너를 통해 제공된다.

의미를 짚자면, Seedream 5.0 Pro는 오픈AI의 GPT-Image 2 등과 직접 겨루는 '프로덕션(실무 투입) 지향' 이미지 모델로 볼 수 있다. 다국어 텍스트와 레이아웃 정밀도를 앞세워 마케팅·디자인 실무 워크플로우를 겨냥한 점이 특징이다. 다만 성능·품질에 관한 서술은 현재 바이트댄스의 자체 발표에 근거한 것으로, 독립적인 검증은 아직 이뤄지지 않았다. 우위 주장은 후속 벤치마크로 확인될 사안인 만큼, 이번 호에서는 기능 사실 위주로 정리한다.

발생일: 2026년 7월 9일. 위 항목과 마찬가지로 72시간 이내 미보도 항목으로 편입했다.
