이번 주 공개된 논문들은 공통된 방향을 가리킨다: AI가 외부 도움 없이 스스로를 개선하는 방법. 이미 알고 있는 지식을 더 잘 꺼내고, 추론 과정을 스스로 평가하고, 이해와 생성을 하나의 모델로 통합하며, 연구 결과를 서로 검증하는 — 자기 완결적 AI를 향한 연구 4편을 분석한다.

강화학습, LLM의 잠자던 지식을 깨우다

논문: Beyond Reasoning: Reinforcement Learning Unlocks Parametric Knowledge in LLMs 저자: Wanli Yang, Hongyu Zang 외 6인 공개일: 2026년 5월 8일

강화학습(RL)은 그동안 AI의 수학적 추론이나 코딩 능력을 높이는 도구로 여겨졌다. 이 논문은 RL이 단순한 '사실 기억'에서도 큰 효과를 발휘한다는 점을 처음으로 체계적으로 입증했다.

연구진은 추론 능력의 개선이 아닌 순수한 지식 회상 능력만 측정하기 위해 통제된 실험 환경을 설계했다. 단계적 사고 과정(chain-of-thought) 없이, 사전 예시 없이, 외부 자료 참조 없이, 한 단계만 거치는 질의응답 설정이다. 보상 함수도 '정답이면 1, 오답이면 0'이라는 가장 단순한 형태를 사용했다.

세 가지 모델 패밀리, 다양한 사실 기반 질의응답 벤치마크에서 RL은 평균 약 27%의 상대적 성능 향상을 달성했다.

가장 흥미로운 발견은 작동 원리에 있다. RL은 새로운 사실을 학습하지 않는다. 모델이 이미 알고 있지만 낮은 확률로 배치되어 있던 정답을 높은 확률 영역으로 '재배치'한다. 창고 깊숙이 있던 책을 눈에 잘 띄는 서가로 옮기는 것과 같다.

특히 학습 데이터의 약 18%에 해당하는 '가장 어려운' 예제들이 전체 성능 향상의 약 83%를 이끌었다. RL이 가장 취약한 지식 영역에서 가장 큰 효과를 발휘한다는 뜻이다.

이 연구는 RL의 활용 범위를 근본적으로 넓힌다. '생각하는 방법'뿐 아니라 '알고 있는 것을 제대로 표현하는 능력'까지 개선할 수 있음이 입증되었다. 의료, 법률, 과학 분야 등 정확한 사실 회상이 중요한 AI 응용에서 RL 기반 후처리의 새로운 가능성을 열어준다. 다만 한 단계 추론에 한정된 실험이므로, 다단계 추론에서의 효과는 추가 검증이 필요하다.

인간 라벨링 없이 AI의 추론 과정을 단계별로 평가하다

논문: Unsupervised Process Reward Models 저자: Artyom Gadetsky, Maxim Kodryan 외 3인 공개일: 2026년 5월 11일

AI가 수학 문제를 풀 때, 최종 답만 맞으면 충분할까? 중간 과정에 오류가 있었다면 그것은 '운 좋은 정답'에 불과하다. 프로세스 보상 모델(PRM)은 추론의 각 단계를 채점하여 어디서 잘못되었는지 파악한다. 수학 시험의 풀이 과정 채점과 같다.

문제는 비용이다. PRM을 학습시키려면 전문가가 각 추론 단계마다 '올바름/틀림'을 직접 표시해야 한다. 확장이 매우 어렵다. 대안으로 다른 LLM이 평가하는 'LLM-as-a-Judge' 방식이 쓰이지만, 정확도에 한계가 있었다.

연구진은 uPRM(unsupervised PRM)을 제안했다. 핵심 아이디어는 언어 모델 자체의 토큰 확률을 활용하는 것이다. 모델이 특정 추론 단계를 생성할 때의 '확신도'를 분석하여, 확신 없이 생성한 단계일수록 오류 가능성이 높다고 판단한다. 별도의 인간 라벨링이나 추가 학습이 필요 없다.

비교 항목uPRM기존 LLM-as-a-Judge
첫 번째 오류 단계 식별 정확도최대 +15%p기준선
다수결 투표 대비 성능 향상최대 +6.9%—
인간 라벨링 필요 여부불필요불필요

ProcessBench 벤치마크에서 uPRM은 LLM-as-a-Judge 대비 최대 15%포인트의 절대 정확도 향상을 달성했다. 테스트 시간 스케일링(추론 시 더 많은 계산 자원을 투입하는 기법)에서는 지도 학습 기반 모델과 비교 가능한 성능을 보였다.

uPRM을 강화학습의 보상 신호로 사용하면, 지도 학습 기반 보상 모델보다 안정적인 정책 최적화가 가능하다. 라벨링 비용이 높은 도메인에서도 효과적인 RL 학습이 가능해진다.

현재는 주로 수학적 추론에서 검증되었으며, 자연어 추론이나 코드 생성 등 다른 도메인에서의 일반화는 향후 과제다.

이해와 생성을 하나의 모델에 — SenseNova-U1의 통합 멀티모달 아키텍처

논문: SenseNova-U1: Unifying Multimodal Understanding and Generation with NEO-unify Architecture 저자: Haiwen Diao 외 57명 (SenseTime Research) 공개일: 2026년 5월 12일

지금까지 AI의 '시각 이해'와 '이미지 생성'은 별개의 모델이 담당했다. GPT-4V가 이미지를 분석하고, DALL-E가 이미지를 만드는 식이다. SenseNova-U1은 이 두 가지 능력을 하나의 모델에 통합했다.

분리된 접근법은 두 가지 문제가 있다. 파이프라인 간 정보 손실이 발생하고, 이해와 생성이 서로의 학습을 돕지 못한다. 인간은 무언가를 그림으로 표현할 수 있을 때 더 잘 이해하고, 잘 이해할수록 더 정확하게 표현한다. SenseNova-U1의 NEO-unify 아키텍처는 이 시너지를 AI에서 구현한다.

연구팀은 두 가지 규모의 모델을 개발했다. 80억 파라미터(8B) 밀집 모델과, 300억 파라미터(30B) 규모의 혼합 전문가 모델(MoE)이다. MoE는 주문에 맞는 전문 셰프만 나서서 요리하는 방식으로, 한 번에 30억 파라미터만 활성화하여 효율성을 확보한다.

주요 성과는 다음과 같다:

  • 텍스트 이해: 텍스트 전용 모델과 비교 가능한 자연어 이해 성능
  • 시각-언어 인식: 이미지 설명, 시각적 질의응답에서 강력한 성능
  • 이미지 합성: 의미적 일관성과 시각적 품질을 모두 갖춘 생성
  • 인포그래픽 생성: 텍스트가 풍부한 정보 시각물 생성 가능
  • 공간 지능: 공간적 이해와 판단이 필요한 과제에서도 능력 발휘

모든 능력이 별도 모듈이 아닌 하나의 통합 모델에서 나온다. 기존의 여러 모델을 연쇄 연결하는 방식과 근본적으로 다르다. 비전-언어-행동과 환경 시뮬레이션 시나리오에서도 초기 역량을 보여, 로봇 공학으로의 확장 가능성도 시사했다.

다만 8B/30B 규모 모델이 GPT-4o급 대형 모델과 어떻게 비교되는지, 특정 벤치마크에서의 정밀 비교는 추가 분석이 필요하다.

AI가 연구하고, 다른 AI가 검증한다 — ARIS의 적대적 협업

논문: ARIS: Autonomous Research via Adversarial Multi-Agent Collaboration 저자: Ruofeng Yang, Yongcan Li, Shuai Li 공개일: 2026년 5월 4일 코드: 오픈소스 공개

AI 에이전트가 스스로 연구를 수행하는 '자율 연구' 시도가 활발하다. 그러나 에이전트가 장시간 자율 실행될수록 근거가 불완전하거나 왜곡된 주장이 생산될 위험이 커진다. AI 연구원이 혼자 일하면 자신의 실수를 스스로 발견하기 어렵다. 인간 연구에서 동료 심사(peer review)가 존재하는 이유와 같다.

ARIS는 세 개 계층으로 구성된다:

  1. 실행 계층: 65개 이상의 재사용 가능한 기술(데이터 수집, 분석, 시각화, 문헌 검색 등)로 실제 연구 작업을 수행한다.
  2. 오케스트레이션 계층: 5가지 종단간 워크플로우를 제공하며, 업무 흐름을 관리한다.
  3. 보증 계층: ARIS의 가장 혁신적인 부분이다. 3단계 증거 검증(무결성 검사, 결과-주장 매핑, 주장 감사), 5단계 편집 파이프라인, 수학적 증명 검증, PDF 시각 검사를 수행한다.

핵심은 '교차 모델 적대적 협업'이다. 한 AI 모델이 연구를 진행하면, 다른 모델 패밀리의 AI가 비판적으로 검토한다. 동일 모델은 같은 편향이나 오류를 공유할 수 있으므로, 다른 기관의 전문가에게 검증을 받는 것과 같은 원리다.

ARIS는 오픈소스로 공개되어 누구나 사용할 수 있으며, HuggingFace에서 112개 이상의 추천을 받았다. 검토자의 승인 메커니즘을 통한 자기 개선 루프를 구현하여, AI 에이전트 시스템 전반에 적용할 수 있는 설계 패턴을 제시한다.

다만 ARIS가 생산하는 연구의 실제 학술적 품질에 대한 인간 전문가의 체계적 평가가 아직 부족하다. 적대적 검증의 비용 효율성 분석도 향후 과제다.
