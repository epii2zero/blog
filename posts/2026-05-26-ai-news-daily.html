1. Apple, genai.apple.com 도메인 등록 — WWDC 2026 AI 대전환 예고

카테고리: 제품/플랫폼

Apple이 WWDC 2026(6/8 기조연설)을 2주 앞두고 genai.apple.com 서브도메인을 등록한 사실이 5/23 DNS 기록에서 발견됐다. 현재 접속하면 타임아웃이 발생하며, 아직 서버 구성이 완료되지 않은 상태다. 도메인 이름이 '생성형 AI(Gen AI)'를 직접 표방하고 있어, Apple Intelligence의 대대적 리브랜딩 또는 별도 Gen AI 플랫폼 출범 가능성이 제기된다.

WWDC에서는 2년간 지연된 Siri 대개편이 마침내 실현될 전망이다. 개인 맥락 인식, 화면 이해, 앱 간 딥 액션 등 ChatGPT/Claude 수준의 대화형 AI 어시스턴트로의 전환이 예상된다. 독립 앱 형태의 챗봇 인터페이스 도입 가능성도 있다.

Bloomberg의 Mark Gurman에 따르면, Apple은 Google에 연간 약 $1B(약 1.4조 원)을 지불하며 1.2조 파라미터 규모의 커스텀 Gemini 모델을 Siri에 탑재할 예정이다. 2026년 1월에 체결된 다년 비독점 계약이다. iOS 27에서는 서드파티 AI 서비스 선택 기능, Image Playground/Genmoji 개선, Visual Intelligence 카메라 통합, 비디오 자동 캡션 등이 예상된다.

Apple은 AI 경쟁에서 뒤처졌다는 평가를 받아왔으나, Google Gemini 파트너십과 자체 Apple Foundation Models의 조합으로 반격을 준비하고 있다. 전 세계 20억+ iPhone 사용자 기반을 고려하면, AI 어시스턴트 시장의 판도 변화가 예상된다.

2. GitHub 내부 3,800개 리포 유출 — Nx Console 공급망 공격

카테고리: 보안

GitHub가 5/20 내부 저장소 약 3,800개가 유출됐음을 공식 확인했다. 공격 벡터는 VS Code 확장 프로그램 Nx Console(nrwl.angular-console)의 트로이목마 버전(v18.95.0)이었다. 5/18 12

~12
UTC 사이 단 11분간 VS Code Marketplace에 게시됐지만, VS Code의 자동 업데이트로 인해 해당 시간에 워크스페이스를 연 개발자들이 감염됐다.

공격 기술: Nx 기여자의 GitHub PAT(개인 액세스 토큰)가 별도의 공급망 공격에서 탈취되어, 리포 푸시 권한과 VS Code Marketplace 게시 자격(VSCE_PAT)이 유출됐다. 2,777바이트의 악성 코드가 워크스페이스 오픈 시 자동 실행되며, install-mcp-extension이라는 위장된 VS Code Task로 Bun 런타임을 통해 498KB 난독화 스크립트를 실행했다.

크리덴셜 수집 범위: 1Password 볼트, npm 크리덴셜(.npmrc), AWS IMDS/IAM, GitHub 토큰(ghp_, gho_, ghs_), GCP 서비스 계정, Docker 레지스트리, Claude Code 설정(~/.claude/ 디렉토리)까지 수집했다. 유출 경로는 OIDC 서명 HTTPS, GitHub API dead-drop, DNS 터널링 3중 경로를 사용했으며, macOS LaunchAgent로 시간당 폴링하는 Python 백도어도 설치됐다.

피해 조직: OpenAI(직원 2명 디바이스 침해, 소스코드 리포 접근), Mistral AI(개발자 1명 엔드포인트 침해, 5GB 소스코드 유출 협박), European Commission, Mercor 등이 포함됐다. 공격 주체 TeamPCP(UNC6780)는 유출 데이터를 $50K 이상에 매각 중이며, 9주간의 조직적 캠페인을 수행해왔다.

이 공격은 CVE 없이 성공했다. 서명된 provenance를 가진 패키지도 악성일 수 있으며, 전통적인 취약성 스캐닝만으로는 현대 공급망 공격을 탐지할 수 없다는 점이 핵심 교훈이다.

3. 엔터프라이즈 AI 비용 위기 — 'Tokenmaxxing'과 에이전트 비용 역전

카테고리: 산업/비즈니스

Fortune과 Tom's Hardware가 5/22 보도한 엔터프라이즈 AI 비용 위기가 업계 전반에 경종을 울리고 있다. 핵심 이슈는 AI 에이전트 사용 비용이 인간 직원 고용 비용을 초과하는 역전 현상이 빅테크 기업에서 속출하고 있다는 점이다.

주요 사례: Microsoft는 6개월 전 수천 명의 직원에게 Claude Code 라이선스를 배포했으나, 비용이 급증하면서 대부분의 라이선스를 취소하고 내부 GitHub Copilot CLI로 전환 중이다. Uber CTO Praveen Neppalli Naga는 "2026년 연간 AI 코딩 도구 예산을 4개월 만에 전부 소진했다"고 밝혔다. Amazon 직원들 사이에서는 가능한 한 많은 AI 토큰을 소비하는 행태를 뜻하는 신조어 'Tokenmaxxing'이 등장했고, Meta에서도 유사한 'Claudeonomics' 리더보드가 운영되고 있다. NVIDIA 응용 딥러닝 VP Bryan Catanzaro는 "우리 팀에서 컴퓨트 비용이 직원 비용을 훨씬 초과한다"고 말했다.

구조적 원인 — 제본스 역설(Jevons Paradox): 토큰 단가 자체는 하락하고 있다. Gartner에 따르면 2030년까지 1조 파라미터 LLM 추론 비용이 2025년 대비 90% 감소할 전망이다. 그러나 에이전트 AI는 표준 LLM 쿼리 대비 최대 1,000배 토큰을 소비하며, Goldman Sachs는 2030년까지 글로벌 월간 토큰 소비량이 120경(quadrillion) 개에 달해 24배 증가할 것으로 예측한다. 단가 하락이 사용량 폭증으로 이어져 총비용이 오히려 증가하는 역설적 구조다.

토큰 단가 인하만으로는 엔터프라이즈 AI 경제성이 확보되지 않으며, 아키텍처 효율화와 사용량 거버넌스가 핵심 과제로 부상했다. Microsoft의 Claude Code 라이선스 취소는 AI 도구 도입의 '허니문 페이즈' 종료를 상징적으로 보여준다.

4. NextEra Energy, $67B에 Dominion Energy 인수 — AI 전력 인프라 메가딜

카테고리: 산업/인프라

NextEra Energy가 5/18 Dominion Energy를 $67B(약 91조 원) 규모의 전액 주식 교환으로 인수한다고 발표했다. 1998년 Exxon-Mobil 합병 이래 최대 에너지 업계 딜이며, 합병 완료 시 세계 최대 규제 유틸리티 기업이 탄생한다.

딜 구조: Dominion 시가총액 $54.3B 대비 23% 프리미엄이 적용됐다. 합병 후 지분은 NextEra 주주 약 74.5%, Dominion 주주 약 25.5%가 된다. 합병 기업 엔터프라이즈 밸류는 $420B으로 미국 3위 에너지 기업이 된다. 규제 승인 후 12~18개월(2027년 예상)에 완료되며, Dominion 고객(버지니아, 노스캐롤라이나, 사우스캐롤라이나)에게 2년간 $2.25B 요금 크레딧을 제공한다. 주주 비과세 구조다.

AI 인프라 핵심 전략: Dominion은 단독으로 51GW 이상의 계약된 데이터센터 용량을 보유하고 있으며, 고객에는 Amazon, Microsoft, Alphabet, Meta, Equinix, CoreWeave가 포함된다. 합병 후 파이프라인은 130GW+ 대형 부하 기회(미국 1억 가구 전력 공급 가능 규모)로 확대되며, 연말까지 40개 데이터센터 캠퍼스를 목표로 한다. Google용 Iowa Duane Arnold 원전 재가동, Meta용 다개 주 2.5GW 태양광/배터리 프로젝트, Dominion 해상풍력 프로젝트 등이 주요 프로젝트다. Trump 무역 딜과 연계하여 텍사스와 펜실베이니아에 미일 정부 공동 소유 10GW급 데이터센터 허브 2곳도 추진 중이다($550B 무역 딜 일환).

CEO John Ketchum은 "미국은 변곡점에 있다. 전력 수요가 세대 단위로 볼 때 유례없이 증가하고 있다"고 밝혔다.

AI 컴퓨팅 인프라에서 전력이 최대 병목임을 극적으로 보여주는 딜이다. 데이터센터 전력 수요가 2030년대 후반까지 피크 수요의 2배 증가가 예상되는 상황에서, 유틸리티 업계의 대형 합종연횡이 AI 시대의 새로운 트렌드로 부상했다. 발표 후 NextEra 주가는 약 5% 하락, Dominion 주가는 9% 상승했다.

5. Penn 대학, 빛-물질 하이브리드 입자 생성 — AI 컴퓨팅 에너지 혁신

카테고리: 연구/하드웨어

펜실베이니아 대학 물리학팀(Bo Zhen 교수 주도)이 AI 컴퓨팅의 에너지 효율을 획기적으로 개선할 수 있는 빛-물질 하이브리드 입자를 생성했다. 연구 결과는 Physical Review Letters에 게재됐다(DOI: 10.1103/gc15-qsvf).

기술 핵심 — 엑시톤-폴라리톤(exciton-polariton): 광자(photon)와 전자(electron)가 원자 수준 두께의 반도체 내부에서 결합하여 형성되는 준입자다. 빛의 속도와 물질의 강한 상호작용을 동시에 구현한다. 나노스케일 캐비티에 빛을 입사시키면, 원자 수준 두께의 반도체 물질과 상호작용하여 엑시톤-폴라리톤이 생성되며, 이를 통해 컴퓨팅에 필수적인 신호 스위칭 로직을 수행한다.

에너지 효율: 약 4x10^-15 줄이라는 극미량의 에너지로 전광(all-light) 스위칭을 시연했다. 소형 LED 점등에 필요한 에너지보다 훨씬 적은 수준이다.

AI 컴퓨팅 적용: 현재 포토닉 AI 칩은 비선형 활성화 단계(결정 연산)에서 광신호를 전자 신호로 다시 변환해야 하며, 이 과정이 속도를 저하시키고 에너지를 소비한다. 이번 돌파구는 전자 변환 없이 순수 광학 처리를 가능하게 하여, 대규모 AI 시스템의 전력 수요를 크게 줄일 수 있다.

AI 훈련 및 추론에 필요한 전력이 기하급수적으로 증가하는 상황에서(위 NextEra-Dominion 딜 참조), 컴퓨팅 아키텍처 자체의 에너지 효율 혁신이 장기적 해법으로 주목된다. 다만 실험실 단계에서 상용화까지의 구체적 타임라인은 아직 제시되지 않았다.
